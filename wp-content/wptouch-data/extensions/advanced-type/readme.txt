Extension Name: Advanced Type
Extension URI: http://www.wptouch.com/extensions/advanced-type
Author: BraveNewCode Inc.
Description: Enables advanced web font support for the full range of Google Fonts and your own sets from Typekit and Fontdeck.
Version: 1.4.1
Depends-on: 4.0

== Long Description ==

Unlock a whole new level of customization with Advanced Typography. Transform your mobile website with custom fonts, chosen by you. Select your fonts from the complete Google Font library or load your sets from Typekit or Fontdeck. Need additional character sets? Go beyond the default with selectable font subsets through the Google Font library.
